# 🔐 SOC Threat Triage Simulator

An interactive cybersecurity simulation built for 
aspiring and practicing SOC analysts.

## 🎯 What Is This?

A browser-based SIEM alert triage game where you think 
like a real SOC analyst. You receive 10 incoming security 
alerts and must classify each one as:

- ⚠️ REAL THREAT — escalate and respond
- ✓ FALSE POSITIVE — benign activity, stand down

Every decision is scored and explained with real-world 
SOC context and MITRE ATT&CK technique references.

## 🧠 What You'll Practice

- SIEM alert triage and prioritization
- Distinguishing real IOCs from noise
- MITRE ATT&CK framework techniques
- Attack chain analysis (kill chain timeline)
- Reducing alert fatigue through context

## 🚨 Alert Scenarios Include

- PowerShell encoded command execution
- LSASS credential dumping (Mimikatz behavior)
- DNS tunneling to C2 server
- Impossible travel / account compromise
- Scheduled task persistence
- Brute force login attacks
- Living off the Land (LotL) techniques

## 📊 Features

- Live scoring and accuracy tracker
- Per-alert feedback with MITRE ATT&CK mapping
- Interactive attack chain kill chain timeline
- Clickable MITRE ATT&CK technique map
- Final analyst grade (S+ to F)
- LinkedIn share button with your score

## 🛠️ Built With

- Vanilla HTML / CSS / JavaScript
- Zero dependencies — runs in any browser
- Mobile responsive
